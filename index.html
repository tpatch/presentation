  6. Get better with tools

    “Sure, the guy in the $6,000 suit... Come on!”

    • Prototype your code: Mockingbird (http://gomockingbird.com/), jsFiddle (http://jsfiddle.net/)
    • Build your code: LESS (http://lesscss.org/), Zen Coding (http://code.google.com/p/zen-coding/)
    • Improve your code: JSLint (http://jslint.com/), JavaScript Compressor (http://www.minifyjavascript.com/), Minify CSS (http://www.minifycss.com/)
    • Version your code: Subversion (http://subversion.tigris.org/), git (http://git-scm.com/), Mercurial (http://mercurial.selenic.com/)
    • Open Web Tools Directory (http://tools.mozilla.com/)

    See season 2, episode 6: Afternoon Delight (http://www.imdb.com/title/tt0515207/)
    Photo: whiteforge

  8. Keep it Local

    “Don't call it that”

    • Get a laptop or tablet (*wink*)
    • Get a good IDE
    • Install a web server
    • Install your languages of choice
    • Develop; as often as you can, wherever you are
    • Extra credit: synchronize your web root
