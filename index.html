What are the features of this framework?

This framework has some great features:

  • Contains three basic layouts: title, bullets, and questions
  • Offers basic keyboard shortcuts: previous slide, next slide, refresh page, and show shortcuts
  • Can easily link to other slides for smooth scroll linking
  • Degrades gracefully

Pretty much every slide can contain footnotes.
